# Resonance Front-End

Utilisation de Shadcn UI, MagicUI, TypeScript et Tailwind CSS.

## Lancer le projet

Lancez le serveur de développement :

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur pour voir le projet en action.
